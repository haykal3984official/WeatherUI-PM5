# WeatherUI

A simple PocketMine-MP plugin that adds a GUI for changing weather in your server. No more typing commands - just click and go!

## Features

- Clean and simple UI using in-game forms
- Three weather options: Clear, Rain, and Thunder
- Works with vanilla Minecraft weather commands
- Lightweight - no bloat, just works

## Requirements

- PocketMine-MP 4.x or 5.x
- FormAPI by jojoe77777 (included in most servers)

## Installation

1. Download the latest release
2. Drop it in your `plugins/` folder
3. Restart your server
4. Done!

## How to Use

Just type `/weatherui` in-game and pick your weather. That's it.

**Note:** This command only works for players, not console.

## Permissions

| Permission | Default | Description |
|------------|---------|-------------|
| `weatherui.command` | OP | Access to /weatherui command |
